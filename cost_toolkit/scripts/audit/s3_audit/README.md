# cost_toolkit.scripts.audit.s3_audit

Comprehensive S3 bucket analysis, cost estimation, and optimization recommendations.

## Modules

- `bucket_analysis.py` - Bucket metadata collection and object analysis
- `cli.py` - Main entry point for the S3 audit CLI
- `constants.py` - Storage class pricing and threshold definitions
- `recommendations.py` - Cost-saving and security optimization recommendations
- `reporting.py` - Report generation and display of analysis results
- `utils.py` - Formatting and calculation helpers

## Usage

Run the CLI to audit S3 buckets with storage class breakdowns, cost estimates, and recommendations.
