# cost_toolkit.scripts.billing

AWS billing and cost reporting scripts.

## Modules

- `aws_hourly_billing_report.py` - Detailed billing for the current hour
- `aws_today_billing_report.py` - Detailed billing for today

## Subpackages

- `billing_report/` - Modular billing report with cost analysis and service checks

## Usage

Run the hourly or daily billing scripts to identify active cost-generating AWS services.
