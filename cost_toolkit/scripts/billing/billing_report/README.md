# cost_toolkit.scripts.billing.billing_report

Modular billing report functionality for AWS cost analysis.

## Modules

- `cli.py` - Main entry point for the billing report CLI
- `cost_analysis.py` - Cost data processing and analysis for billing reports
- `formatting.py` - Report formatting and display functions
- `service_checks.py` - Service status checking functions for billing optimization
- `service_checks_extended.py` - Extended service status checks for additional AWS services

## Usage

Run the CLI to generate combined billing reports with regional breakdowns,
usage details, and service status checks.
