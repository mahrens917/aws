# cost_toolkit.scripts.management

AWS resource management and standardization scripts.

## Modules

- `aws_ebs_volume_manager.py` - EBS volume management including deletion and info retrieval
- `aws_s3_standardization.py` - S3 bucket standardization and configuration
- `aws_volume_cleanup.py` - Volume tagging, snapshot deletion, and S3 bucket listing

## Subpackages

- `ebs_manager/` - Modular EBS volume manager with CLI, operations, and reporting

## Usage

Run individual scripts to manage EBS volumes, standardize S3 buckets, or clean up volumes.
