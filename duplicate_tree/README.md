# duplicate_tree

Duplicate tree analysis package.

## Modules

- `analysis.py` - Duplicate tree detection and analysis logic
- `cache.py` - Cache management for duplicate tree analysis results
- `cli.py` - CLI workflow for duplicate tree analysis
- `core.py` - Duplicate tree helpers focused on exact duplicate detection
- `deletion.py` - Directory deletion logic for duplicate trees
- `workflow.py` - Workflow orchestration for duplicate analysis

## Usage

Run the CLI to scan directory trees, identify exact duplicates, and optionally delete them.
